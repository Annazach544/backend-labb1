# Databasbaserat CV

Detta projekt är skapat som en laboration i serverbaserad webbutveckling.

## Funktioner

- Visa kurser
- Lägga till kurser
- Radera kurser
- Validering av formulärdata

## Tekniker

- Node.js
- Express
- EJS
- SQLite
- CSS

## Starta projektet

```bash
npm install
node database/install.js
node server.js
